# Project Memory

## Core
IronSharp: faith-based daily devotional & accountability app.
5 themes: Vesper (default, dark), Parchment, Sage, Dusk, Slate. No hardcoded colors.
Fonts: Playfair Display (serif, headings/scripture), DM Sans (body/UI).
Lovable Cloud backend. Google SSO + email/password auth.
